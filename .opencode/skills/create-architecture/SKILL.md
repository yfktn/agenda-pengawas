# Create Architecture

Purpose

Generate a technical architecture document from docs/prd.md.

Input

docs/prd.md

Instructions

1. Read docs/prd.md.
2. Identify:
   - frontend requirements
   - backend requirements
   - database requirements
   - integrations
3. Fill architecture template based on skills/create-architecture/templates/architecture-template.md.
4. Do not invent business requirements.
5. Record assumptions separately.

Output

docs/architecture.md
